# Stylebook summary

## Color palette

- Primary: `#7c3aed`
- Primary dark: `#5b21b6`
- Secondary: `#10b981`
- Background: `#f7f7fb`
- Surface: `#ffffff`
- Text: `#1d1031`
- Border: `#e5dff5`
- Error: `#dc2626`

## Typography

- Display headings: **Bebas Neue**
- Body text and UI controls: **Inter**

## UI principles

- High-contrast cards with soft shadows.
- Rounded components (12px default radius).
- Clear visual hierarchy using semantic sections and card grouping.
- Buttons with primary and outline variants for action clarity.
